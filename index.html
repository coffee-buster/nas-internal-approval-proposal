Quick recommendation

For the core setup, start with the Synology DS225+, two NAS-rated 4TB drives in RAID1, a USB-capable UPS, and a simple gigabit switch. The printer is shown separately because it has a separate budget.

RAID1 helps if one drive fails, but it is not a full backup. A separate cloud or offsite backup should still be planned.
